# 💍 Shaadi Bio – Marriage Biodata Generator

A simple and elegant **Marriage Biodata Generator** built with **Next js** and **Tailwind CSS**.
Users can fill in their personal and family details, upload a photo, and instantly generate a **print-ready matrimonial biodata template**.

---

## 🚀 Features

* 📝 Fill biodata using an interactive form
* 🖼 Upload profile photo
* 🎨 Multiple biodata templates
* 📄 Print / Download biodata as PDF
* 📱 Responsive layout
* ⚡ Instant preview of biodata

---

## 🛠 Tech Stack

* **Next.js**
* **Tailwind CSS**
* **JavaScript (ES6)**
* **react-to-print**

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/yourusername/shaadi-bio.git
```

Navigate to project

```
cd shaadi-bio
```

Install dependencies

```
npm install
```

Start development server

```
npm run dev
```

---

## 📄 How It Works

1. User fills biodata form.
2. Data is stored in state.
3. Biodata preview component renders the details inside a template.
4. User can print or download the biodata as a PDF.
